# Changelog

Todos as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere a [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2024-03-24

### Adicionado
- 🎯 Sistema completo de detecção de objetos em tempo real usando YOLOv5
- 🖥️ Interface de usuário interativa com Gradio
- 🎤 Síntese de voz para anunciar objetos detectados
- 📊 Sistema de logging em CSV para registro de detecções
- 💾 Salvamento automático de imagens detectadas
- ⚙️ Sistema de configuração flexível via arquivo .env
- 📦 Estrutura modular e profissional do código
- 🐳 Suporte a Docker e Docker Compose
- 🧪 Testes automatizados com pytest
- 📝 Documentação completa
- 🚀 CI/CD com GitHub Actions

### Melhorias
- Refatoração completa do código original
- Separação de responsabilidades em módulos
- Melhor tratamento de erros e logging
- Interface mais intuitiva e profissional
- Performance otimizada
- Suporte multi-plataforma (Windows, Linux, Mac)

### Corrigido
- Correção de bugs no sistema de áudio
- Melhoria no tratamento de exceções
- Correção de problemas de compatibilidade

### Tecnologia
- Python 3.8+
- YOLOv5 (Ultralytics)
- OpenCV
- Gradio
- gTTS (Google Text-to-Speech)
- PyTorch
- Pandas
